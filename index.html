Projects / selected work

TamZhi — Tamil Transliteration
Paper

Reverse transliteration that converts shorthand Romanized Tamil (with/without vowels) back to Tamil script using a hybrid rules + ML approach.

  • Problem: inconsistent shorthand causes errors in typing tools and NLP tasks.
  • Data: curated transliteration pairs; normalization; noise handling.
  • Approach: rules for vowel omissions + ML model for edge cases.
  • Result: 90%+ accuracy; practical for IMEs and education tools.
Python Flask NLP
